# 儿童识字小报生成器 🎨

一个纯前端的儿童识字小报生成工具，适合 5-9 岁儿童的识字学习。根据用户输入的主题和标题，自动生成带有识字标签的儿童小报图像。

![技术栈](https://img.shields.io/badge/技术栈-HTML%2FCSS%2FJS-green)
![部署](https://img.shields.io/badge/部署-GitHub%20Pages-purple)

## ✨ 功能特点

- 🎨 **本地词汇库** - 内置 6 种常见场景（超市、医院、公园、学校、动物园、图书馆）的词汇数据库
- 🖼️ **智能绘图** - 调用专业图像生成 API，生成儿童绘本风格的小报
- 📝 **识字标签** - 每个物体都带有识字标签，格式：拼音 + 汉字
- 🔒 **隐私保护** - API 密钥存储在浏览器本地，不上传到任何服务器
- 🌐 **零服务器成本** - 纯前端实现，可部署到 GitHub Pages 或直接本地使用
- 📱 **响应式设计** - 支持手机、平板、电脑等多种设备
- ⚡ **快速生成** - 无需等待 AI 生成词汇，直接使用本地词库，速度更快

## 🔑 API 密钥配置

### Nano Banana Pro API Key

用于生成图像。

- 获取地址：https://kie.ai/api-key
- 成本：按实际生成次数付费
- 文档：[ai-docs/api-nano-bana-pro.md](ai-docs/api-nano-bana-pro.md)

**说明**：
- 本应用使用本地词汇数据库，无需 OpenAI API Key
- 只需配置 Kie AI API Key 即可完成全部功能

## 📖 使用说明

### 首次使用

1. 打开网页
2. 在 "API 密钥配置" 区域输入您的 **Nano Banana Pro API Key**
3. 点击 "💾 保存 API 密钥" 按钮
4. 看到 "✅ 已配置" 提示

**说明**：
- 只需配置一个 API 密钥（Kie AI / Nano Banana Pro）
- 词汇使用本地数据库，无需额外配置

### 生成识字小报

1. 在 "主题/场景" 输入框中输入主题（如：超市、医院、公园）
2. 在 "小报标题" 输入框中输入标题（如：《走进超市》）
3. 点击 "🎨 生成识字小报" 按钮
4. 等待 30-60 秒
5. 查看生成的小报图像
6. 点击 "📥 下载图片" 保存到本地

## 🎯 核心流程

```
用户输入（主题 + 标题）
    ↓
从本地数据库获取相关词汇（支持 6 种常见场景）
    ↓
基于模板生成完整的绘图提示词
    ↓
调用 Kie AI API 创建图像生成任务
    ↓
前端轮询任务状态直到完成（每 5 秒查询一次，最多 5 分钟）
    ↓
在页面显示生成的小报图像
    ↓
用户下载图片
```

**支持的主题**：
- 超市、医院、公园、学校、动物园、图书馆
- 其他主题会使用默认通用词汇

## 🎨 技术栈

- **前端**：HTML5 + CSS3 + JavaScript (ES6+)
- **样式**：CSS Grid + Flexbox
- **API**：Kie AI API (nano-banana-pro) - 图像生成
- **词汇库**：本地 JSON 数据库（预定义 6 种场景）
- **存储**：浏览器 localStorage（API 密钥）

## 🐛 常见问题

### 1. 提示 "API 密钥未配置"

**解决方法**：
- 检查是否已正确输入并保存 Kie AI API Key
- 刷新页面，检查状态是否显示 "✅ 已配置"

### 2. 词汇不完整或不准确

**原因**：
- 本应用使用本地词汇数据库，支持常见场景（超市、医院、公园、学校、动物园、图书馆）
- 如果输入的主题不在数据库中，会使用默认通用词汇

**解决方法**：
- 尝试使用支持的主题名称
- 或等待未来版本添加更多场景

### 3. 图像生成失败

**可能原因**：
- Kie AI API Key 不正确
- 账户余额不足
- 提示词包含敏感内容
- 生成任务超时

**解决方法**：
- 检查 API Key 是否正确
- 登录 Kie AI 平台检查余额
- 尝试更换主题或标题
- 如果超时，可以重试

### 4. 跨域（CORS）问题

如果遇到跨域问题，可以考虑：
1. 使用本地浏览器扩展解决
2. 使用 CORS 代理服务
